## Tips for Generation
- Use 1:1 square for icons, tokens, and frames
- Use 16:9 for backgrounds and celebration effects
- Always specify "transparent background" + "no text" + "no logos" to avoid watermarks/garbled text
- Run each prompt 2–3 times and pick the cleanest result
- Consistency across a set matters more than any single image being perfect
